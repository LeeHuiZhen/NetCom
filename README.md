# Network Design for Faculty of Computing (Block N28B)

This repository contains the comprehensive network infrastructure design for the new two-storey Faculty of Computing building (Block N28B) at Universiti Teknologi Malaysia (UTM). The project was developed by **TTL Network Solutions** as part of the SECR 1213 Network Communication course.

## 🚀 Project Overview
The objective was to design a robust, scalable, and secure Local Area Network (LAN) for a specialized academic facility. The design accommodates various specialized labs, classrooms, and administrative offices within a RM 2,000,000 budget.

### Key Facilities Covered:
* Cisco Networking Lab & Embedded Systems Lab
* General Purpose Labs
* Video Conferencing Room & Hybrid Classroom
* Lecturer Offices & Support Spaces

## 📂 Documentation Structure
The project was executed in six distinct phases:

| Phase | Description |
| :--- | :--- |
| **Task 1** | **Architectural Planning:** Floor plans designed using AutoCAD with a focus on optimal spatial arrangement for network nodes. |
| **Task 2** | **Feasibility Assessment:** Technical, economic, and operational analysis confirming the project's viability within budget. |
| **Task 3** | **Hardware Selection:** Strategic selection of enterprise-grade devices (Cisco ISR 1100, Catalyst 9200L, Huawei CloudEngine). |
| **Task 4** | **LAN & WAN Connections:** Physical distribution diagrams using Fiber Optic backbones and CAT6 horizontal cabling. |
| **Task 5** | **IP Addressing:** Implementation of VLSM (Variable Length Subnet Masking) and VLAN segmentation for security. |
| **Task 6A** | **Final Report:** Integrated technical proposal and comprehensive project compilation. |

## 🛠️ Technologies & Tools
* **Design:** AutoCAD, Draw.io
* **Networking:** VLSM, VLAN, LAN/WAN Architecture, SD-WAN
* **Hardware:** Cisco (Routers/Switches/Firewalls), Huawei
